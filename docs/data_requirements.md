# 数据需求说明

这个 Skill 的上限，不取决于文风，而取决于数据层。

## 一、股票分析最低需要的数据

### 1. 基础行情
- 当前价格
- 总市值
- 企业价值 EV
- 52周高低点
- 日均成交额

### 2. 财务质量
- 营收
- EBIT / EBITDA
- 净利润
- 自由现金流
- 每股收益 EPS
- 毛利率 / 营业利润率 / 净利率
- ROE / ROIC
- 资产负债率
- 净负债 / EBITDA
- 利息保障倍数

### 3. 估值
- PE
- PB
- PS
- EV/EBIT
- EV/EBITDA
- FCF Yield

### 4. 稳定性
- 5年营收复合增速
- 5年EPS复合增速
- 5-10年利润波动情况
- 回购/分红记录

### 5. 质性信息
- 商业模式
- 行业结构
- 主要竞争对手
- 管理层资本配置记录
- 关键风险

## 二、基金 / ETF 分析最低需要的数据

- 基金类型
- 跟踪指数或策略
- 费率
- 规模 AUM
- 流动性
- 前十大持仓
- 行业分布
- 国家/地区分布
- 历史收益
- 波动率
- 最大回撤
- 跟踪误差
- 分红政策

## 三、持仓复盘额外需要的数据

- 买入价格
- 买入日期
- 买入理由
- 当前仓位占比
- 计划持有期限
- 能承受的最大回撤

## 四、组合构建额外需要的数据

- 总资金
- 每月新增资金
- 年龄
- 风险承受度
- 投资期限
- 是否需要现金流
- 账户类型
- 税务约束

## 五、数据降级原则

### Level A
数据完整，可以做相对完整判断。

### Level B
数据部分缺失，只能做半定量分析。

### Level C
只有名称或非常少的信息，只能做框架判断与条件式建议。

## 六、严禁伪精确

缺失这些数据时，不得伪装成可以严谨估值：
- 自由现金流
- ROIC / ROE
- 负债压力
- 基金费率与持仓
- 历史波动与最大回撤

## 七、优先补齐顺序

当数据不全时，优先补：
1. 当前价格 / 市值 / 基础估值
2. 财务质量
3. 资产负债表与现金流
4. 行业与竞争格局
5. 基金持仓与费率
6. 回撤与波动
